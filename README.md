# Система цифровой обработки сигналов СКВТ (RDC)

Проект реализует вычислительное ядро для преобразователя «угол-код» на базе ПЛИС Xilinx Spartan-3.

## Характеристики синтеза
- Платформа: Spartan-3 (XC3S1000)
- Тактовая частота (Fmax): 84.72 МГц
- Ресурсы (Slices): 6%
- Ресурсы (DSP/Multipliers): 0% (вычисления реализованы на логических элементах)

## Структура проекта
В папке /rtl представлены исходные коды модулей:
- RDC_top.vhd — верхний уровень иерархии.
- ADC_UPRAV_DEMOD.vhd — захват данных и демодуляция.
- CORDIC.vhd — итерационный алгоритм вычисления угла.
- ROM_ARCTAN.vhd — таблица арктангенсов для CORDIC.
- DDS.vhd — синтезатор опорных частот.
- WIDTH_LUT.vhd — таблица для формирования ШИМ.
- CDC.vhd — модуль синхронизации для асинхронных входов.
- dcm_pitanie.vhd — конфигурация тактового генератора.
- lcd_contr.vhd — контроллер вывода на АЦЖКИ индикатор.
- pitanie_SP3_1000.ucf — файл ограничений.

## Верификация
В папке /tb находится файл TB_CORDIC.vhd. 
Тестбенч реализует подачу входных векторов и автоматическую проверку выходных данных модуля CORDIC через конструкцию assert.
